Prototyping only — not production-ready code

This skill generates prototype-quality HTML & CSS for rapid visual exploration, stakeholder alignment, and design direction validation. Output is not intended for production deployment — it has not been audited for accessibility compliance (WCAG), performance optimization, cross-browser/device testing, framework integration, security review, or production build tooling. Use it to move fast during ideation, then hand off to your engineering team for production implementation with proper testing and review cycles.
