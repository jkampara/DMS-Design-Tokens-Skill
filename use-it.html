Get started

Two ways to use the skill

Point your agent at the hosted URL for instant, zero-install access — or download the zip and set it up locally. Both paths give the agent the same bundle.

Option 1 — Hosted URL

No download, no install. Paste the bundle URL into your first message and the agent fetches tokens, recipes, and docs on demand.

Option 2 — Download & install

One zip (~960 KB) contains everything the agent needs — SKILL.md, recipes, token JSON, CSS, and Tailwind preset. Unzip into your tool's skill folder.

Set up for your tool

Cursor

Unzip into ~/.cursor/skills/. The skill is auto-discovered. Say "use the dms-design-tokens skill" or just "build me a Salesforce hero".

Terminal one-liner 
curl -L "https://sfdc-ui-documentation-667d4c5e8394.herokuapp.com/dms-design-tokens/dms-design-tokens-skill.zip" \
  -o /tmp/sf-skill.zip && \
  unzip -o /tmp/sf-skill.zip -d ~/.cursor/skills/ && \
  rm /tmp/sf-skill.zip

Claude Code

Unzip into ~/.claude/skills/. Type /dms-design-tokens to activate, or just say "build me a Salesforce hero".

Terminal one-liner 
curl -L "https://sfdc-ui-documentation-667d4c5e8394.herokuapp.com/dms-design-tokens/dms-design-tokens-skill.zip" \
  -o /tmp/sf-skill.zip && \
  unzip -o /tmp/sf-skill.zip -d ~/.claude/skills/ && \
  rm /tmp/sf-skill.zip
