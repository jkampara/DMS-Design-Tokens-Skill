The pitch in two cards

What it is, and why we built it

What is it?

A self-contained brand bundle, hosted at a single URL

Every token (color, typography, spacing, grid, radii) and every recipe (hero, card, CTA, page-shell, gradient regions) lives in one published bundle. Agents fetch what they need on demand and follow the rules the bundle ships with. No SDK, no install, no per-project setup — just a URL and a skill entrypoint.

Why does it exist?

So the brand survives the AI page-building workflow

Agents that don't know the brand invent visual values: ad-hoc hex colors, off-grid spacing, wrong type ladders. The skill replaces invention with retrieval — the agent reads the canonical recipe, copies the canonical CSS, and audits its own output against the brand contract before sending. The brand stays the brand, even when nobody on the design team sees the draft.

Why it matters

Built for every role in the workflow

Faster builds, higher fidelity, fewer review cycles. The skill turns "implement the brand refresh" from a multi-week alignment exercise into a fetch-and-follow workflow.

Engineers

Save hours per component

Skip the "which hex? which radius? which spacing step?" research loop. The bundle resolves every visual decision to a single var(--…) lookup — fetch three JSONs, copy the recipe skeleton, ship.

AI agents & copilots

Fewer hallucinations

Every value the agent emits is verified against a machine-readable source of truth. The gaps protocol forces agents to ask instead of guess — no more invented hex codes, made-up font stacks, or plausible-but-wrong spacing.

Designers

A launchpad for new patterns

The bundle ships brand-approved, AA-validated primitives as composable building blocks. Designers can remix them into novel card types, blade layouts, and themed event pages — build something new without starting from zero.

Accessibility reviewers

Pre-built for contrast

Every color pairing is AA-validated (≥ 4.5 : 1) at build time. Background → ink → highlight triples are curated per surface so adopters never accidentally ship a sub-AA combination.

Brand & marketing

Brand-refresh expression, captured

The refreshed identity — gradients, type pairings, Day / Night system, four theming axes — lives in code, not a static PDF. Every page built with the skill ships the new brand automatically.

What to build with it

Ideas you can ship today

The skill isn't limited to copying existing recipes verbatim — it's a toolkit for building new patterns, exploring theme variations, and shipping brand-safe pages at prototype speed.

Rapid page-pattern prototyping

Spin up new page layouts in minutes instead of days. The agent assembles hero, card-grid, body, and CTA blades from recipe skeletons.

New blade types and interaction patterns

Building a blade the recipes don't cover yet? The agent reads token-roles.md to pick the right semantic tokens by intent.

Day / Night themed pages and sections

Scope data-theme on the page, a single section, or a multi-blade region wrapper and the cascade repaints everything.

Accent-surface and gradient region exploration

Explore the 25 sanctioned blade backgrounds — five theme families, each with curated gradient + ink + highlight triples.

Event and campaign microsites

Dreamforce, World Tour, product launches — each needs the brand-refresh expression with campaign-specific theming.

Multi-brand reskinning

Swap the primitive token ladder and every alias inherits — same recipes, different brand.

What it can do

Capabilities at a glance

Three pillars that protect the brand from drift — Theming, Layout, and Quality.

Tokens, themes & surfaces

Four-family token glossary (color, spacing, type, radii), 25 sanctioned blade backgrounds, and per-page or per-blade theme scoping via data-theme.

Dark-mode components

Dual-island cards on Night blades (Day-island default, Night-island opt-in) and an on-dark CTA modifier — cards auto-paint, no extra class needed.

Heroes & grid systems

Hero blade variants (marquee, centered, large, mirror) plus card grids with four math-fit bands that flex-wrap so cards stay centered at every breakpoint.

Content blocks & regions

Featured Quote 3-stage cascade (desktop → tablet → mobile) and contiguous-gradient super-blade regions that span multiple sections.

Layout safety nets

Page-shell overflow prevention, card-grid guardrails (min-width, overflow-wrap, box-sizing, stretch), and headline tier enforcement.

Self-audit & validation

Every generation ends with an 8-question self-audit — gate-guarded checks that catch brand drift before you ship.
